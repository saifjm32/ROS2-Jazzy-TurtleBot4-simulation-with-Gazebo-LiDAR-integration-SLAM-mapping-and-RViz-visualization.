# TurtleBot4 ROS2 SLAM Mapping

## Overview

This project demonstrates the implementation of a TurtleBot4 mobile robot using ROS2 Jazzy and Gazebo simulation.

The robot was equipped with a simulated LiDAR sensor and differential drive system to perform Simultaneous Localization and Mapping (SLAM) in a virtual environment.

## Technologies

- ROS2 Jazzy
- Gazebo
- RViz
- TurtleBot4
- Linux
- SLAM Toolbox

## Key Features

- Robot simulation in Gazebo
- LiDAR integration
- ROS2 topic communication
- Manual robot navigation
- Occupancy grid map generation
- Real-time visualization using RViz
- SLAM implementation
